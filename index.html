📘 راهنمای ورود به سرور BedWars

1. نسخه Java ماین‌کرافت رو اجرا کن (1.8 تا 1.20 پشتیبانی میشه)

2. به بخش Multiplayer برو و روی Add Server کلیک کن

3. آدرس سرور رو وارد کن:

MC_King_aXz.aternos.me:36377

4. وارد شو، تیم انتخاب کن و نبرد رو شروع کن!
